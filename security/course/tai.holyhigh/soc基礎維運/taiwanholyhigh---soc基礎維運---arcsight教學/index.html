    Set Up #

    先進到https://bit.ly/44IFecN下載ArcSight Console，並且全部default下一步

    1. (editor需要admin權限)進入localhost host DNS解析 -> C:\Windows\System32\drivers\etc\host 新增 211.75.237.80 chtpoc
    2. 打開ArcSight Console，用之前提供的帳密以及最後打chtpoc，進行登入
    3. 登入之後的狀態

    ArcSight預設有分大小寫
