    Exploit - New Programming Language #

    1. 查詢歌詞的一些關鍵字會發現rockstar這個網站

    2. 把歌詞貼到decoder會出現一堆數字，猜測應該是ascii code
    3. Decode ascii and get flag: picoCTF{rrrocknrn0113r}
