    Recon #

    這一題有一點通靈，沒有很喜歡

    1. 首先給的cipher很明顯是ascii的hex
    2. 轉換過後也很明顯是base64
    3. 在轉換過後，只有底線後面的部分要轉換成hex
    4. 如果把東西print出來的話會看到flag.txt的字樣，感覺上是一個file的byte code，到file signature去看會發現magic header是一個zip file，uncompress之後就會發現flag.txt
