# Simple PWN 0x21(fopen, fread, fwrite, fclose)
###### tags: `CTF` `PWN` `eductf`

Version: Ubuntu 20.04

## Original Code
:::spoiler fopen
```cpp=
#include <fcntl.h>
#include <stdio.h>

int main()
{
    FILE *fp;
    fp = fopen("./test", "r");
    fclose(fp);

    return 0;
}
```
:::

:::spoiler fread
```cpp=
#include <fcntl.h>
#include <stdio.h>

int main()
{
    FILE *fp;
    char buf[0x10];

    fp = fopen("./test", "r");
    fread(buf, 0x1, 0x10, fp);
    fclose(fp);

    return 0;
}
```
:::

:::spoiler fwrite
```cpp=
#include <fcntl.h>
#include <stdio.h>

int main()
{

    fp = fopen("./test_write", "r");
```
