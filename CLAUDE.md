# Project conventions

## Books Notes posts

Whenever a new post is added under `_posts/Books Notes/**`, after creation:

1. Look up the book online (original edition and, if translated, the Chinese/translated edition) to fill in the frontmatter fields already present in the post body: 作者, 出版社, 出版日期, 譯者, 譯版出版社, 譯版出版日期.
2. Populate the `tags:` array in the frontmatter with 1–5 tags. Always include the author, publisher, translator, and translated-edition publisher as tags (when applicable). Feel free to add topical/genre tags too — look at sibling posts in the same category for existing tag conventions, and invent new tags when appropriate.
